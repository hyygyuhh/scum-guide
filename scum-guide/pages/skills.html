四大核心属性

SCUM的角色系统基于四大核心属性，每个属性直接影响角色的基础能力和对应技能的学习速度。合理分配属性点是生存的关键。

力量 (STR)
  • 加成：伤害、负重
  • 影响技能：近战武器、重武器、弓箭
  • 建议：喜欢近战和负重多的玩家优先
体质 (CON)
  • 加成：耐力、血量、恢复速度
  • 影响技能：奔跑、游泳、抗性
  • 建议：新手推荐点满奔跑，提升机动性
敏捷 (DEX)
  • 加成：制作速度、行动速度
  • 影响技能：潜行、偷窃、制作、陷阱
  • 建议：喜欢潜行和制作的玩家优先
智力 (INT)
  • 加成：学习速度、维修能力
  • 影响技能：机械维修、电子维修、医疗
  • 建议：喜欢载具和基地建设的玩家优先

推荐加点方案

根据不同的游玩风格，以下是三种经过验证的加点方案。属性点总数有限，合理分配至关重要。

独狼生存流
  • 属性分配：STR 3 / CON 4 / DEX 5 / INT 2
  • 核心思路：高敏捷保证潜行和制作，高体质保证机动性
  • 适合：喜欢单人游玩、避免正面冲突的玩家
战斗突击流
  • 属性分配：STR 6 / CON 4 / DEX 3 / INT 1
  • 核心思路：高力量保证近战伤害，适合PvP
  • 适合：喜欢正面战斗的玩家
基地建设流
  • 属性分配：STR 2 / CON 3 / DEX 3 / INT 6
  • 核心思路：高智力加速维修和制作
  • 适合：喜欢建造和团队支持的玩家

技能升级技巧

SCUM中的技能通过实际使用来提升，不同属性对应的技能有不同的升级方式。以下是各属性技能的高效升级方法。

  • 力量技能：负重行走、使用近战武器攻击僵尸或玩家
  • 体质技能：持续奔跑、游泳、承受伤害（被僵尸攻击后恢复）
  • 敏捷技能：潜行移动（蹲伏靠近僵尸而不被发现）、制作物品、开锁
  • 智力技能：修理物品（武器、防具、载具）、阅读书籍、使用医疗用品

重要提示

在分配属性点之前，请务必了解以下关键信息，避免浪费属性点。

  • 重生时属性点重置，但技能经验保留 -- 可以放心尝试不同加点方案
  • 新建角色建议至少点1级Survival，否则无法生火，初期生存会非常困难
  • 属性影响技能升级速度，高属性学对应技能更快 -- 属性与技能要匹配
  • PvP服务器建议平衡发展，不要偏科 -- 综合能力比单项极限更重要
